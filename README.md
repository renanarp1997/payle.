# Payle — site (Next.js)

Landing da Payle em Next.js 15 (App Router): checkout inteligente, integrações e dois temas visuais.

## Rodar localmente

```bash
npm install
npm run dev
```

- **Tema escuro (azul + verde):** [http://localhost:3000](http://localhost:3000)
- **Tema claro (branco + azul):** [http://localhost:3000/branco-azul](http://localhost:3000/branco-azul)

## Build

```bash
npm run build
npm start
```

## Estrutura

- `app/` — rotas e layout
- `components/payle/` — landing, ícones SVG e tokens de tema (`payleTheme.ts`)
